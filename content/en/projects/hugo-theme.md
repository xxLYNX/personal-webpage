## Overview

A clean, minimalistic theme for Hugo with a focus on readability, simplicity, and multilingual support. Perfect for personal portfolios, blogs, and documentation sites.

## Features

- Responsive design that works on all devices
- Dark/light mode toggle with system preference detection
- Multilingual support (English, Spanish, French)
- Fast loading times with optimized assets
- SEO-friendly with proper meta tags
- Syntax highlighting for code blocks
- Social media integration

## Technical Details

Built using Hugo's powerful templating system and styled with Tailwind CSS for a modern, utility-first approach. The theme leverages Hugo's content organization features to provide a flexible structure for different content types.

## Design Philosophy

The theme follows a "content-first" philosophy, ensuring that the reader's focus remains on the content without unnecessary distractions. Typography choices prioritize readability, and spacing is carefully considered throughout.

## Usage

The theme is designed to be easy to customize through Hugo's configuration system. Colors, fonts, and layout options can be modified through simple TOML configuration files.
